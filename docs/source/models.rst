Supported Models
================

We support all safetensor formatted models on `HuggingFace Hub <https://huggingface.co/>`_ for the following architectures. 

.. list-table::
   :header-rows: 1

   * - Model
     - Size
     - HuggingFace Handle Example
   * - Gemma 2
     - 2B, 9B, 27B
     - google/gemma-2-2b
   * - Llama 3
     - 8B, 27B, 405B
     - meta-llama/Llama-3.1-8B
